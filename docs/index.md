**Robot Twin**

**Sim2Real**

**Real2Sim**

## 快速开始

```bash
# 克隆仓库
git clone https://github.com/FennMai/IS_Bot.git
cd IS_Bot

# 安装依赖
pip install -r requirements.txt

```
